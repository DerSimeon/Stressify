# Security Policy

## Intended use

Stressify is intended for:
- Testing Minecraft plugins
- Debugging and developing servers
- Controlled load testing on servers you own or are authorized to test

Stressify is not intended for:
- Attacking public servers
- Denial-of-service attacks
- Harassment or griefing

## Reporting issues

If you discover a security issue or misuse vector:
- Report it privately to the maintainer
- Do not publish exploit details without coordination

## Scope

In scope:
- Unsafe defaults
- Resource exhaustion bugs
- Packaging or update risks

Out of scope:
- Misuse by users acting against stated intent
